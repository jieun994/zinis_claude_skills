# 나답게 (jieun) — 신지은님 분신 에이전트 모음

신지은님처럼 일하는 **분신 에이전트**를 모아둔 플러그인. 상황에 맞는 "나"를 불러 일을 맡길 수 있음.

## 1. 무엇을 하나요

- 신지은님의 일하는 방식(간결한 커뮤니케이션·책임 범위 분리·방어적 설계)을 담은 에이전트
- 현재 3명 수록

- **메일 쓰는 나 (`jieun-mail`)** — 상사 보고 메일을 신지은님 말투로 작성
- **기획하는 나 (`jieun-planner`)** — IA·WBS·플로우차트·주간보고 등 기획 산출물을 신지은님 기준으로 작성·검토
- **개발 통역하는 나 (`jieun-dev-translator`)** — API 명세·ERD·기술문서를 기획자 눈으로 통역, 여러 자료를 이어보며 대화

## 2. 어떻게 쓰나요

- Claude에게 평소처럼 말하면 알맞은 "나"를 자동 호출

- 메일: "이 내용 상사한테 보고 메일로 써줘"
- 기획 검토: "이 플로우차트 신지은님 관점으로 검토해줘"
- 개발 통역: "이 API랑 저 ERD 연결해서 같이 보자"

## 3. 예시

| 이렇게 말하면 | 이런 결과 |
|---|---|
| "이번 주 진행상황 상사 보고 메일 써줘" | 짧고 핵심만 담은 보고 메일 초안 + 제목 제안 |
| "이 IA에서 빠진 화면 없는지 봐줘" | 예외·권한·흐름의 빈틈 지적 + 보완안 제시 |
| "개발자한테 받은 이 문서 기획 입장에서 봐줘" | 쉬운 말 통역 + 개발자에게 물을 질문 정리 |
