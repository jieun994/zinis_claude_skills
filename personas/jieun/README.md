# 나답게 (jieun) — 신지은님의 분신 에이전트 모음

신지은님처럼 일하는 **에이전트(분신)**들을 모아둔 플러그인입니다.
상황에 맞는 "나"를 불러서 일을 맡길 수 있어요.

## 뭐 하는 플러그인인가요?

신지은님의 일하는 방식(간결한 커뮤니케이션, 책임 범위 분리, 방어적 설계)을
담은 에이전트들입니다. 현재 두 명이 들어 있어요.

- **메일 쓰는 나 (`jieun-mail`)** — 상사 보고 메일을 신지은님 말투로 써줍니다.
- **기획하는 나 (`jieun-planner`)** — IA·WBS·플로우차트·주간보고 같은 기획 산출물을
  신지은님 기준으로 만들고 검토합니다.

## 어떻게 쓰나요?

Claude에게 평소처럼 말하면 됩니다. 알맞은 "나"를 알아서 불러옵니다.

- 메일: **"이 내용 상사한테 보고 메일로 써줘"**
- 기획 검토: **"이 플로우차트 신지은님 관점으로 검토해줘"**

## 예시

| 이렇게 말하면 | 이런 일이 일어나요 |
|---|---|
| "이번 주 진행상황 상사 보고 메일 써줘" | 짧고 핵심만 담은 보고 메일 초안 + 제목 제안 |
| "이 IA에서 빠진 화면 없는지 봐줘" | 예외·권한·흐름의 빈틈을 짚어주고 보완안 제시 |
