# 회의록 작성 스킬 (meeting-minutes)

## 뭐 하는 스킬인가요?

회의 녹취록이나 메모를 주면, 표준 엑셀 양식의 회의록을 만들어주는 스킬입니다.
회의록을 한 번도 써본 적 없어도 괜찮아요 — Claude가 "녹취록 있으세요?"부터 하나씩 물어보며 순서대로 끌고 갑니다.

단순 요약이 아니라, 실무에서 회의록 때문에 가장 자주 문제가 되는 세 가지를 자동으로 챙깁니다:

1. **서로 다르게 알고 있던 것** — 양측이 다르게 이해하고 있던 안건은 "누가 어떻게 알고 있었는지"를 나눠서 적고, 회의에서 "그 자료는 무시하세요"라고 한 부분은 어떤 파일의 어떤 부분인지까지 남깁니다.
2. **"검토해볼게요"의 이유** — 상대방이 검토하겠다고 한 항목엔 왜 검토하는지 배경을 같이 적어서, 나중에 정확한 답이 돌아오게 합니다.
3. **할 일은 하나씩 쪼개서** — "정리해서 전달드릴게요"를 한 줄로 뭉치지 않고, 전달물 하나당 한 줄씩 담당자·기한·형식까지 나눠 적습니다.

그리고 **회의에서 안 나온 내용은 절대 지어내지 않습니다.** 모르는 담당자·날짜는 "미정(확인 필요)"로 표시되고, 마지막에 "이것들만 확인받으면 완성"이라는 목록을 따로 줍니다.

## 어떻게 쓰나요?

Claude에게 이렇게 말하세요:

> "회의록 작성해줘" / "이 녹취록 회의록으로 정리해줘" / "오늘 미팅 내용 회의록으로 만들어줘"

그러면 Claude가 순서대로 물어봅니다:

1. **파일 주세요** — 녹취록(클로바노트 txt/docx 등) 하나면 시작할 수 있어요. 녹취록이 없으면 기억나는 내용을 대화창에 적어도 됩니다.
2. **기본 정보 확인** — 회의명·날짜·참석자를 Claude가 알아서 뽑아 보여주니, 틀린 것만 고쳐주세요.
3. **초안 확인** — 엑셀로 만들기 전에 내용을 먼저 보여줘요. "여기는 회의에서 언급이 없어 비워뒀어요" 같은 안내와 함께요.
4. **완성** — 확인해주시면 엑셀 파일이 만들어집니다.

### 같이 주면 더 좋은 파일

| 파일 | 효과 |
|------|------|
| 녹취록 (필수에 가까움) | 회의 내용·할 일의 근거 |
| 본인 메모 | 녹취에 안 잡힌 맥락 보완 |
| 회의에서 같이 본 자료 (PPT/엑셀) | "이 자료의 이 부분은 무효" 기록 가능 |
| 직전 회의록 | 지난 결정이 뒤집힌 부분 감지 |

## 예시

**입력:** "어제 그린마켓 킥오프 회의 녹취록이야. 회의록으로 정리해줘" + 녹취록.txt

**결과:** `[그린마켓 앱 리뉴얼] 회의록_20260703.xlsx`

| 구분 | 내용 (일부) |
|------|------------|
| 검색 | [이해 차이 확인] 검색 처리 구조 ・고객사: AI 통합 판단 구조로 전제 ・수행사: 유형별 분리 매핑으로 전제 ・회의 결과: 고객사 전제로 확인, 설계 문서 수정 필요 |
| 정책 | 포인트 정책: 내부 검토 후 전달 예정 ・검토 사유: 통합회원 전환 시 기존 포인트 처리에 법무 판단이 필요하기 때문 |

TO-DO는 이렇게 쪼개져 나옵니다:

| 항목 | 영역 | 내용 | 완료희망일 | 담당자 |
|------|------|------|-----------|--------|
| 정책 | 고객사 | 보안가이드 전달 - 형식: PDF | 2026-07-10 | 홍길동 |
| 정책 | 고객사 | AI 운영 가이드 전달 - 형식: 초안 가능 | 2026-07-10 | 홍길동 |
| 플로우 | 고객사 | 플로우 영역 구분본 전달 - 형식: AI/API 색상 구분 | 미정(확인 필요) | 김철수 |
