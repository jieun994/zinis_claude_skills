## 1. 무엇을 하나요

- 프로젝트의 **작업 분해 구조(WBS)를 엑셀로 자동 생성**
- 프로젝트 내용만 설명하면 작업을 단계별(대분류→세부작업, L1~L3)로 분해
- 일정·담당자 제안 → 엑셀로 정리
- 간트차트·진척률·공휴일은 자동 생성

## 2. 어떻게 쓰나요

- Claude에게 **"OO 프로젝트 WBS 만들어줘"** 라고 요청
- Claude가 아래 순서로 단계적으로 질문 (한 번에 다 묻지 않음)

1. 프로젝트명 · 기간(시작~종료) · 한 줄 개요
2. 적용 단계(L1) 선택 → 단계별 **중분류(L2)** 제안 → 확인
3. 각 L2의 **세부작업(L3)** 제안 → 확인
4. 작업별 **일정·담당자** 배정 → 확인 → 엑셀 생성

- 각 단계마다 초안을 먼저 제시 → 빼고 더하기만 하면 됨
- 참고할 기획서·요건 문서는 첨부하거나 내용을 붙여넣으면 더 정확히 반영

## 3. 만들어지는 결과

엑셀 시트 6개 (표지 → 사용안내 → 개정이력 → 요약 → WBS → 공휴일)

- **표지** — 프로젝트명·기간·작성자·버전
- **사용안내** — 파일 보는 법·수정법 (온보딩)
- **개정이력** — 버전·날짜·작성자·변경 내용 (최초 작성 행 자동)
- **요약** — 전체 + 단계(L1)별 기간·목표진척율·진척율·상태 (WBS와 자동 연결)
- **WBS** — 작업표(No·L1~L3·담당자·산출물·일정·진척률·상태·비고) + 자동 간트차트
- **공휴일** — 프로젝트 기간 내 대한민국 공휴일 자동 목록

결과 미리보기: [samples/결과_예시.xlsx](samples/결과_예시.xlsx)

## 4. 작동 규칙

대부분 자동이라 몰라도 되지만, 참고용으로 정리.

### 작업 레벨과 No
- **L1**(대분류) → **L2**(중분류) → **L3**(작업, 말단) 3단계
- L1은 **8개 고정**: `프로젝트 관리` `기획` `디자인` `퍼블리싱` `개발` `테스트` `오픈` `안정화` (해당 단계만 사용)
- No는 자동(3칸, 각 자릿수가 레벨) — 예) 기획=`100`, 요구사항분석=`110`, 수집=`111`
- 행 색: L1=짙은 회색 · L2=연노랑 · L3=흰색. 전체 작업 행 최대 60개

### 사용자 입력 vs 자동
| 레벨 | 직접 입력 | 자동 계산 |
| --- | --- | --- |
| L3(말단) | 시작일·종료일·진척율·상태 | 진행일·작업량·목표진척율 |
| L1·L2 | 이름·상태만 | 작업량·진척율 전부 |

- **진행일** = 시작~종료 사이 평일 수(공휴일 제외)
- **총작업량** = L3는 진행일, L2는 하위 L3 합, L1은 하위 L2 합
- **목표작업량** = 총작업량 × (기준일이 일정 중 어디쯤인지 가중치)
- **진척율** = L3는 직접 입력(%), L1·L2는 실/총 자동

### 상태와 간트 색
- 상태(드롭다운): `대기` `진행중` `고객검토` `수정반영` `완료` `보류`
- 완료=초록 / 진행중·고객검토·수정반영=진척만큼 파랑+잔여 연파랑 / 대기=회청 / 보류=주황

### 일정·기준일·공휴일 (자동)
- 시작일·종료일로 간트 막대 자동 생성, 상위(L1·L2) 기간은 자식들의 최소~최대로 자동 설정
- 시트에서 날짜를 바꾸면 달력·간트도 따라옴
- 기준일(오늘 선) = 오늘 날짜 자동, 간트에 세로선으로 표시
- 공휴일 = 대한민국 공휴일 자동 반영 (음력·대체휴일 포함, 제헌절 제외)

## 5. 엑셀에서 직접 수정할 때

생성된 WBS는 대부분 수식으로 자동 계산됨. 직접 손볼 때 참고.

### 안전 — 입력 칸만 수정 (수식 안 건드림)
- **일정 변경** — 해당 작업의 `시작일·종료일`만 변경 → 진행일·작업량·간트 자동 갱신
- **진행률 변경** — **말단(L3)** 작업의 `진척율` 칸에 % 입력(예: 50%). 상위는 자동
- **상태 변경** — `상태` 칸 드롭다운 선택 → 간트 색 변경
- 전체 시작·종료일 변경 — 맨 위 메타의 시작일·종료일 셀 수정 (달력 따라옴)

### 주의 — 작업(행)을 추가·삭제할 때
- 수식이 레벨마다 다르고, 상위 작업이 특정 자식 칸을 더하는 방식이라 손이 필요함
- **L3 추가** — 같은 레벨 행을 복사→붙여넣기 하면 수식 그대로 따라옴 (직접 넣을 것: `시작일·종료일·진척율·상태`)
- **상위(L1·L2)의 `총작업량`·`실작업량`은 "자식 합" 수식**(예: `=O9+O10`) — 자식을 추가/삭제하면 합 수식에 새 행 포함하도록 수정(예: `=O9+O10+O11`)
- 행 **삭제** 시 엑셀이 참조를 자동 정리하지만, 상위 합계 수식이 맞는지 한 번 확인

### 레벨별 수식 요약
| 칸 | L3(말단) | L1·L2(상위) |
|---|---|---|
| 총작업량 | `=진행일` | 자식들의 합 |
| 실작업량 | `=총작업량 × 진척율` | 자식들의 합 |
| 진척율 | 직접 입력(%) | `=실작업량 ÷ 총작업량` |
| 진행일·목표작업량·목표진척율 | 모든 레벨 자동 수식 (건드릴 일 없음) | |

- 요약: **날짜·진척율·상태만 바꾸는 건 자유**, **행을 추가/삭제할 때만** 상위 합계 수식 확인
