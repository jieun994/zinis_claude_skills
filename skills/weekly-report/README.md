# 주간보고서 자동 작성 (weekly-report)

## 1. 무엇을 하나요

- 매주 작성하는 **주간업무 보고서(PPT)** 를 기존 양식 그대로 자동 작성
- **W1**(평소 주간보고)과 **W2**(월말 마지막 주 — 뒤에 "월간 결산" 4장 추가) 중 선택
- `지난주 실적 / 이번주 목표 / 이번주 실적 / 차주 목표`(누적 %)만 입력하면 **진척률 표 자동 계산**
- **지난주 보고서 파일 첨부** 시 전체 일정·누적 진척률·업무구분 이어받고, 바뀐 부분만 갱신

## 2. 어떻게 쓰나요

- Claude에게 아래처럼 요청

- "이번주 주간보고서 만들어줘"
- "지난주 보고서 첨부했어. 이번주 W2로 작성해줘" (월말 마지막 주)
- "주간 실적 보고서 만들어줘"

Claude가 아래 순서로 질문:

1. **W1 / W2** 중 선택 (월말이면 W2 추천)
2. **지난주 보고서 파일** 여부 (없으면 빈 양식으로 시작)
3. **내용 입력** — 구분별 진척률 숫자 4개, 이번주 실적 / 차주 계획 / 이슈 등

- 입력 완료 시 양식 그대로 채워진 **PPT 파일** 생성, "이번주 누적 진척률 N%" 요약 제공

## 3. 진척률 계산 방식

- 구분(전체/기획/개발 등)마다 누적 % 4개 입력 → 보고서 표 6칸 자동 계산

| 표의 칸 | 계산식 | 예시(지난주 2.1 / 이번주목표 7.3 / 이번주실적 6.9 / 차주목표 11.8) |
|---|---|---|
| 주간 계획 | 이번주 목표 − 지난주 실적 | 5.2% |
| 주간 실적 | 이번주 실적 − 지난주 실적 | 4.8% |
| 누적 계획 | 이번주 목표 | 7.3% |
| 누적 실적 | 이번주 실적 | 6.9% |
| 차주 주간 | 차주 목표 − 이번주 실적 | 4.9% |
| 차주 누적 | 차주 목표 | 11.8% |

## 4. 입력 예시

```
W2로 만들어줘. 지난주 보고서는 weekly_W1.pptx 야.

[진척률 누적 %]
전체:  지난주 2.1 / 이번주목표 7.3 / 이번주실적 6.9 / 차주목표 11.8
기획:  지난주 3.7 / 이번주목표 6.5 / 이번주실적 7.0 / 차주목표 19.0
개발:  지난주 2.9 / 이번주목표 22.3 / 이번주실적 20.5 / 차주목표 29.0

[이번주 실적 / 차주 계획]
프로젝트 관리 - 이번주: WBS 현행화, 주간보고 / 차주: 요구사항 확정 대기
개발 - 이번주: 인터페이스 분석 / 차주: 코어 API 설계

[이슈] 요청: 요구사항 확정 / 이슈: 없음
```

## 5. 결과 예시

`samples/` 폴더에서 실제 결과물 확인 가능

- `주간보고서_결과예시_W1.pptx` — 평소 주간보고(8장)
- `주간보고서_결과예시_W2.pptx` — 월말 보고(12장, 월간 결산 포함)

## 6. 참고

- 결과는 **PowerPoint(.pptx)** — 열어서 바로 검토·결재 가능
- 진척률은 입력값으로 계산되니, 입력값이 맞는지 한 번 확인
- 빈 양식(`assets/W1_blank.pptx`, `assets/W2_blank.pptx`)은 회사 데이터 없는 깨끗한 시작 템플릿
